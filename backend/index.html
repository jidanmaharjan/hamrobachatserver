Login User

type: POST

link: https://hamrobachat.herokuapp.com/api/v1/login

body: { "email": "example@example.com", "password": "password" }

Logout User

type: GET

link: https://hamrobachat.herokuapp.com/api/v1/logout

Profile of Logged-In User

type: GET

link: https://hamrobachat.herokuapp.com/api/v1/profile

Bachat API

Create Monthly Bachat from current month

type: GET

link: https://hamrobachat.herokuapp.com/api/v1/create

Get monthly data

type: GET

link: https://hamrobachat.herokuapp.com/api/v1/month/:month

description: :month is parameter. put date as 'MM YYYY' for data. Example: .../month/09 2022
